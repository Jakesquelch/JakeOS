# JakeOS
JakeOS is a personal productivity web application designed to help you manage your daily life through three core modules:

1. **To-Do List** - Task management with priorities and grouping
2. **Journal** - Personal journaling (planned feature)
3. **Habit Tracker** - Track daily habits and build consistency (planned feature)

### Technology Stack
- **Frontend Framework:** Angular 21 (latest version)
- **Language:** TypeScript 5.9
- **Styling:** Custom CSS
- **Build Tool:** Angular CLI
- **Package Manager:** npm 11.6.4

### Required Software:
- **Node.js** (v18 or higher recommended)
- **npm** (v11.6.4 or higher) - comes with Node.js
- **Git** (for version control)

### Setup:
```bash
cd frontend
npm install
npm start
```

This will:
- Start the Angular development server
- Compile the TypeScript code
- Open the application at `http://localhost:4200`
- Enable hot module replacement (auto-refresh on code changes)
