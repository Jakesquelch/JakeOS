- Our whole app is displayed on the [index.html](../frontend/src/index.html) file
- We then have our app dir a later below. We have [app.ts](../frontend/src/app/app.ts) which is the main application shell (this is our whole app). We have some ts for setting the colour theme here.
- Our app.ts file imports RouterOutlet. This is what is displayed in our [app.html](../frontend/src/app/app.html) page. So whenever we click around on the app we will be updating our route, this will get the html code for that page, then display it. We also have our left side nav with the theme footer showing here constantly. So when we update the router outlet, it is just the content section that is being updated.
- So we have a default route in [app.route.ts](../frontend/src/app/app.routes.ts) which means this is the page you will go to when you go to the webpage/our app. That route is the [Dashboard](../frontend/src/app/pages/dashboard/dashboard.ts). So if you have a look at the html for the Dashboard component. You can see we have our summary/dashboard page displayed there.
